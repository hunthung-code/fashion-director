# ECHO TAIPEI — Demo Brand

> This is a fictional brand created to demonstrate the fashion-director plugin end-to-end. All brand details are invented.

**ECHO TAIPEI** is a contemporary Taiwanese streetwear label targeting urban youth aged 20–28. It blends Taipei's underground music scene with quality basics and limited-edition drops.

---

## How to use this example

This folder acts as a `{client}-fashion-config/` layer for ECHO TAIPEI. To run the plugin against it, add a reference in your CLAUDE.md:

```markdown
Brand config: examples/echo-taipei/
```

Or copy the folder to your project as `echo-taipei-config/` and reference it from there.

---

## Demo workflow — all 8 skills in sequence

Run these in order to see the full plugin output for ECHO TAIPEI:

```
# 1. Audit the brand's current health
/brand-audit 幫我診斷 ECHO TAIPEI 的品牌現況

# 2. Get FW2026 trend intelligence
/trend-intelligence 掃描 FW2026 亞太街頭風格主要趨勢

# 3. Build a curated seasonal package
/seasonal-curation 幫 ECHO TAIPEI 做 FW2026 趨勢包

# 4. Set the creative direction for this season
/creative-direction 幫 ECHO TAIPEI 定調 FW2026 視覺方向

# 5. Get specialist input for the streetwear category
/style-specialist ECHO TAIPEI 的 FW2026 系列需要 street-hype 視角

# 6. Check the market opportunity
/market-strategy 台灣街頭時裝市場分析，ECHO TAIPEI 的定位機會

# 7. Generate the client lookbook
/lookbook-generator 把 ECHO TAIPEI FW2026 的 4 個造型做成 lookbook PDF

# 8. Package everything for the client
/output-generator 把今天所有 ECHO TAIPEI 的輸出打包成 Gamma deck
```
