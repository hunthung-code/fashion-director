## Lookbook image direction

When generating image placeholder descriptions for `lookbook-generator`, use these environment cues:

| Look type | Environment description |
|---|---|
| Core basics (tee, trousers) | 台北市停車場 / Taipei parking garage, late afternoon light |
| Outerwear | 大安區巷弄 / Da'an alley, overcast |
| Accessories / detail shots | 咖啡桌面 / coffee table surface, natural light |
| Seasonal editorial spread | 捷運站月台 / MRT platform, motion blur background |
| Group looks | 地下展場 / underground venue before doors open |

## Tone reminders for all outputs

- No superlatives
- Quantities over adjectives ("200 units" not "limited edition")
- City-specific (Da'an / Zhongshan / Xinyi — not "Taipei" as generic, not "Asia")
- Short sentences
- Never use "amazing," "perfect," "best," "guaranteed"
